About HireFormer

HireFormer is a Transformer-based talent-matching prototype that turns resumes into 768-d semantic embeddings using sentence-transformers/all-mpnet-base-v2 and ranks candidates against any job description by cosine similarity.

What's inside

  • Recruiter — paste a JD → ranked resumes from your indexed pool
  • Applicant — upload a resume → predicted domain, gap analysis vs a target JD, feedback, and matching jobs
  • Find Jobs — searches the real-world JD dataset (job_title_des.csv)
  • Dataset — visualises the indexed resume pool by category

Backend: FastAPI + sentence-transformers + scikit-learn · Frontend: vanilla JS + Tailwind + Chart.js
